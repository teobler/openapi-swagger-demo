# What for

this is a demo about OpenAPI 3.0 to provide a schema to generate front-end request code by using [redux-action-generator](https://github.com/teobler/redux-action-generator/tree/openapi-3.0)

## Start

1. clone code to your local
2. run `./gradlew boot run`
3. access http://localhost:8080/api-docs in your browser you can get OpenAPI 3.0 schema
4. access http://localhost:8080/swagger-ui/index.html?configUrl=/api-docs/swagger-config in your browser you can see swagger v3
